Meet NSS : Cohort VI

A little bit about the class.

After six months of training, thousand of lines of code, and gallons of coffee, we're ready for the coding challenges of tomorrow.

The Web Developer Bootcamp is a six-month, full-time program. The curriculum teaches the skills and technologies required to become a junior full-stack software developer. Cohort Six focused on a full JavaScript stack, building the frontend chops to create websites, web apps, and hybrid mobile applications with HTML, CSS, JavaScript, jQuery, AngularJS, and Ionic. We laid down a solid foundation in general software engineering and server­-side development concepts, e.g. test­-driven development, object­-oriented programming, pair programming, source code control in a team environment, common code patterns, NoSQL and SQL databases with NodeJS, Express, Hapi, MongoDB, and Postgresql.
